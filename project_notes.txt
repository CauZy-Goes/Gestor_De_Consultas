1 - Start

    pip install django
    pip install pillow = imagens
    django-admin startproject core . =  criar o core
    python manage.py runserver = iniciar server
    python manage.py startapp pacientes   == criar app app

2 - Banco

    python manage.py makemigrations
    python manage.py migrate

3 -  Aplicacao

    python manage.py createsuperuser
